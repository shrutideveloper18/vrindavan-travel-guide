# Vrindavan Travel Guide 

A simple and responsive travel guide website about **Vrindavan**, built using **HTML and CSS**. This project highlights famous temples, best time to visit, festivals, and useful information for visitors.

---

##  Live Demo

https://shrutideveloper18.github.io/vrindavan-travel-guide/

---


## Features

* Clean and responsive layout
* Sections for **Description**, **Places to Visit**, **Best Time to Visit**, and **Festivals**
* Card-based design for better readability
* Beginner-friendly code structure

---

## Places Covered

* Banke Bihari Temple
* ISKCON Temple (Krishna Balaram Mandir)
* Prem Mandir
